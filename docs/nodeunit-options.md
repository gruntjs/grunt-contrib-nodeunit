# Settings

## options.reporter
* Type: `String`  
* Default: `'grunt'`  

Specifies the reporter you want to use. For example, `default`, `verbose` or `tap`.

## options.reporterOutput
* Type: `Boolean`  
* Default: `false`  

Specifies the file the `reporter`'s output should be saved to. For example, `tests.tap`.
